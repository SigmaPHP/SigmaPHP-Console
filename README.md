# SigmaPHP-Console

Build powerful, feature-rich command-line applications with an elegant and intuitive API.
Simplify argument parsing, command routing, interactive prompts, and terminal output.
Designed for performance, extensibility, and a first-class developer experience.

## Installation

``` 
composer require sigmaphp/sigmaphp-console
```

## Documentation

### Create new Console App

To create a new console app, you can start from the provided template by copy it and rename it
as you like:

```
cp bin/app.php.template my-app

chmod +x my-app

./my-app
```
before running the app, make sure to edit, add your commands and other functions.
